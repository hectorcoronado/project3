An online morse code decoder (and eventually encoder as well!).

About This Project

Fairly simple to use (times are in milliseconds):

  1. Tap on button 1 for less than 500ms to add a dot (the one on the middle of the board).
  2. Tap on it for more than 500ms for a dash.
  3. If you're done making a letter, tap the 2nd buton (on the edge of the board).
  4. If you've entered a complete word, hold button 2 for 1000ms.
  5. If you want to decode what you've entered, hold button 1 for 3000ms.
